# 🚫 Gambling Awareness & Support Tool

A web-based application designed to help individuals stay strong in their commitment to avoid gambling and betting activities.

## 🌟 Features

### 1. **Commitment Tracker**
- Daily check-in system to track your progress
- Streak counter showing consecutive days of staying strong
- Visual feedback and motivation

### 2. **Content Filter Helper**
- **URL Checker**: Verify if websites are gambling-related before visiting
- **Text Analyzer**: Check text content for gambling-related keywords
- Instant warnings and recommendations

### 3. **Support Resources**
- Direct links to gambling addiction helplines
- Recovery-focused mobile app recommendations
- Educational resources and support groups

### 4. **Daily Motivation**
- Rotating motivational messages
- Positive reinforcement system
- Focus on building a better future

## 🚀 How to Use

### Getting Started
1. Open `index.html` in your web browser
2. Start your journey by clicking "Today's Check-in"
3. Use the content filters to check URLs and text before engaging
4. Explore support resources when you need help

### Daily Routine
- **Morning**: Check in for the day and read your motivation
- **Throughout the day**: Use URL/text checkers when unsure about content
- **Evening**: Reflect on your progress and stay committed

### Keyboard Shortcuts
- `Ctrl + K`: Focus on URL input field
- `Ctrl + T`: Focus on text input field
