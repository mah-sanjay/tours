# NDIS Provider Pro - Use Case Diagrams

## Use Case Diagram Notation
- **Actor**: External entity that interacts with the system
- **Use Case**: Functionality provided by the system
- **System Boundary**: Boundary of the NDIS Provider Pro system
- **Association**: Relationship between actor and use case
- **Include**: One use case includes another use case
- **Extend**: One use case extends another use case
